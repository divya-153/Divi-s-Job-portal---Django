# WalkInHub – Walk-In Job Portal

## Overview
WalkInHub is a full-stack web application designed to connect job seekers with recruiters through walk-in job opportunities. The platform enables recruiters to post job openings while allowing job seekers to search jobs and share interview experiences.

## Features
- User registration and login authentication
- Role-based dashboards for job seekers and recruiters
- Recruiters can post and manage job opportunities
- Job seekers can search and filter job listings
- Interview experience sharing and review system
- Cloud-based media storage

## Technologies Used
Frontend: HTML, CSS, JavaScript  
Backend: Python, Django  
Database: MySQL  
Deployment: Render  
Media Storage: Cloudinary

## Installation
1. Clone the repository
2. Install dependencies
3. Run Django server
